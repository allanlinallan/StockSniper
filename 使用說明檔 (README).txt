[1] 日常操作流程 (Daily Routine)
-------------------------------------------------------------------
本系統設計為「一鍵操作」，請依照以下順序執行：

步驟 A：更新每日數據 (盤中 9:00 - 13:30 或 收盤後)
   👉 點擊桌面捷徑或執行 `Update_Data.bat`
   
   說明：
   1. 系統會掃描全台上市股票。
   2. 自動計算技術指標 (距低點%、均線乖離)。
   3. 針對符合條件的股票，自動爬取 Google 新聞進行 AI 簡易分析。
   4. 結果會存成 `sniper_report_news_日期.csv`。
   * 注意：若出現 IP 冷卻訊息為正常現象，系統會自動休息 60 秒後繼續。

步驟 B：開啟戰情室 (查看結果)
   👉 點擊桌面捷徑或執行 `Run_Dashboard.bat`
   
   說明：
   1. 瀏覽器會自動開啟戰情室介面。
   2. 您可以使用左側篩選器過濾股票。

[2] 訊號解讀指南 (Signal Dictionary)
-------------------------------------------------------------------

🚀 突破新高 (Breakout)
   - 定義：股價創下近 200 日新高。
   - 策略：強勢股順勢操作。注意成交量是否放大，嚴設停損。

🔥 即將創高 (Near High)
   - 定義：距離歷史高點 < 5%。
   - 策略：蓄勢待發，關注是否能一舉突破。

⚡ 底部翻揚 (Rebound) ★★★ 最推薦
   - 定義：股價在低檔區 (距低點 < 15%) 且站上 5 日均線。
   - 策略：高勝率買點。代表主力開始點火，且下檔風險有限。

🟢 歷史極低 (Historical Low)
   - 定義：股價在近 200 日最低點附近 (距低點 < 5%)。
   - 策略：左側交易 (存股族最愛)。適合分批佈局，但需忍受盤整。

💤 低檔盤整 (Consolidation)
   - 定義：股價在低檔，但還沒站上均線。
   - 策略：觀察名單。等待轉成「⚡」訊號再進場效率較高。

[3] 常見問題排除 (FAQ)
-------------------------------------------------------------------
Q1: 執行 Update_Data.bat 時出現紅字錯誤？
A1: 
  - 若是 "Connection aborted"，代表被證交所暫時擋 IP，程式會自動處理，不用擔心。
  - 若是 "ModuleNotFoundError"，請檢查是否有點兩下啟動 Conda 環境。

Q2: 戰情室介面顯示 "找不到報表"？
A2: 請先執行步驟 A (Update_Data.bat) 產生今日報表。

Q3: 如何篩選出「基本面」好的股票？
A3: 在戰情室左側的「新聞關鍵字」欄位輸入：營收, 獲利, 成長。
    系統會篩選出新聞標題帶有這些利多字眼的股票。
